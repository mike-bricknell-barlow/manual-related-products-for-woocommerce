Allows you to choose a manual set of products for display under the Related Products section on WooCommerce single product pages.

== Description ==

This plugin allows you to take manual control over the products that are shown under the Related Products section on WooCommerce single product pages. When editing a product, it's possible to pick up to 4 products which will be shown in the Related Products section. Choosing less than 4, or not choosing any, will show the default selection of related products according to woocommerce rules to fill the 4 slots.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Install the plugin through the WordPress plugins screen directly, or download the .zip file, and upload it by going to Plugins -> Add New -> Upload .zip file.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Edit a product, and go to the new Related Products tab in the product settings to choose your related products.

== Settings ==

None - no configuration required.
